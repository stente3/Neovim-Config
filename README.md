# LunarVim Configuration with Lua

This is my custom LunarVim setup using Lua. 
It includes a variety of plugins to enhance productivity, development, and workflow automation.

## Included Plugins

| Plugin                                                            | Functionality                     |
| ----------------------------------------------------------------- | --------------------------------- |
| [oil.nvim](https://github.com/stevearc/oil.nvim)                  | File management with a simple UI. |
| [harpoon](https://github.com/ThePrimeagen/harpoon)                | Quick file navigation.            |
| [hop.nvim](https://github.com/phaazon/hop.nvim)                   | Fast navigation within buffers.   |
| [nvim-spectre](https://github.com/nvim-pack/nvim-spectre)         | Search and replace tool.          |
| [trouble.nvim](https://github.com/folke/trouble.nvim)             | Diagnostics list.                 |
| [todo-comments.nvim](https://github.com/folke/todo-comments.nvim) | Highlight and manage TODOs.       |
| [autosplit.nvim](https://github.com/yochem/autosplit.nvim)        | Auto-resizing window splits.      |
